# 🤖 AI-Powered Forex Trading Agent for XAUUSD

An advanced AI trading assistant focused on XAUUSD (Gold) trading, designed to improve decision-making, discipline, and consistency for prop firm and personal trading accounts.

---

## 🎯 Project Objective

Develop a personal AI trading assistant that acts as:

- 📊 Market Analyst
- 📘 Trading Journal Coach
- 🧠 Psychology Mentor
- 🛡️ Risk Manager
- 🏢 Prop Firm Compliance Monitor
- 📈 Performance Dashboard
- 🚀 Daily 1% Improvement Coach

---

## ✨ Core Features

### 📊 Market Analysis Engine
- Analyze XAUUSD price action
- Detect support and resistance
- Identify market structure (BOS, CHoCH)
- Detect liquidity sweeps
- Generate trade scenarios

### 📘 Trade Journal Analysis
- Import MT5 trade history (CSV)
- Calculate win rate
- Calculate risk-to-reward ratio
- Calculate expectancy
- Detect recurring mistakes

### 🧠 Trading Psychology Tracker
- Record emotions before and after trades
- Detect revenge trading
- Detect fear and greed patterns
- Generate mindset recommendations

### 🛡️ Risk Management Engine
- Lot size calculator
- Daily loss monitoring
- Overall drawdown monitoring
- Position sizing

### 🏢 Prop Firm Rule Manager
- FTMO, The5ers, Atmos Fund compatibility
- Daily loss rule monitoring
- Maximum loss rule monitoring
- Best day rule monitoring

### 🎯 Daily Readiness Checklist
- Sleep quality
- Stress level
- Focus level
- Confidence level
- Trading readiness score

### 📈 Discipline Dashboard
- Rule adherence score
- Habit completion score
- Discipline trend tracking

### 📓 Thought Journal Analyzer
- Analyze daily thoughts
- Detect recurring patterns
- Generate weekly insights

### 🚀 1% Improvement Engine
- Suggest one high-impact improvement every day

### 📷 Chart Screenshot Analyzer
- Analyze TradingView and MT5 screenshots

### 🔔 Smart Alert System
- Telegram alerts
- Email notifications
- Risk and psychology alerts
